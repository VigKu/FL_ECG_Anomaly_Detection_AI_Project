Python Files:
	1) balance_mihbih_train_datatset: balance MIH-BIH train dataset
	2) balance_preproc_mihbih_test_dataset: balance pre-processed MIH-BIH test dataset
	3) preprocess_mihbih_train_val_dataset: pre-process MIH-BIH train and validation dataset
	4) split_mihbih_train_val_dataset: split original MIH-BIH train data into train and validation datasets
	5) test_pre_trained_model: test script for models on test dataset
	6) tf_attn_model: modified model with attention network architecture in functional API
	7) tf_base_model: base model architecture in functional API
	8) train: Re-train base model with best hyper-parameters
	9) train_attn: Re-train modified model with attention networks with best hyper-parameters
	10) train_tune: tune hyper-parameters for base model
	11) train_tune_attn: tune hyper-parameters for modified model with attention networks
	12) utils: contain custom made python functions
